# 초보 개발자의 팀 프로젝트를 도와주는 Mini Github Web  
- 기존 Git의 사용법과 비슷하지만 좀 더 초보자들에게 편리한 팀 프로젝트용 게시판을 구현해봤습니다.<br>  
- 팀장과 팀원들로 구성이 되어있고 초대 및 추방 기능 존재<br>  
- 코드 병합 과정 및 충돌하는 경우 색상으로 표시되는 기능 존재<br>  
- 다양한 커뮤니티 활동이 가능한 게시판 기능 존재<br>
